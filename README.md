# QuoteVault-Android
QuoteVault is a modern quote discovery and personalization app built using Kotlin and Jetpack Compose, with Supabase as the backend.

This project was developed as part of a mobile application developer assignment, with a strong focus on AI-assisted development and rapid iteration.

## Tech stack
- Kotlin
- Jetpack Compose
- MVVM Architecture
- Supabase (Auth + Database)
- Room (offline caching)
- DataStore (settings)
- WorkManager (daily quote)
- Android App Widget

## Features (Planned)
- User authentication
- Quote discovery and categories
- Favourites and collections
- Quote of the day + notifications
- Sharing and quote cards
- Personalization & themes
- Home screen widget
